Foi desenvolvido uma página web para a comparação de um carro elétrico com um carro a combustão.

Para fazer os testes é necessário definir os valores a partir dos sliders e clicar no botão "Comparar". Os valores iniciais definidos por eles já são os especificados no exemplo dado em https://educacaoautomotiva.com/2018/10/12/calculo-consumo-carro-eletrico/.

Feito o teste, a comparação é exibida pelos gráficos, o primeiro exibindo o quanto aquele carro elétrico percorre pelo litro definido no carro a combustão, exibindo que sua eficiência é bem maior. O segundo exibe o quanto um carro elétrico e um carro a combustão gastaria para percorrer 100km com os valores definidos.
